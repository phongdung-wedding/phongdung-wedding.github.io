Đặt file nhạc nền tại đây với tên: one-life.mp3

- Đường dẫn đầy đủ khi chạy web: /wedding/music/one-life.mp3
- Định dạng nên dùng: MP3 (hoặc .m4a/.ogg — nếu đổi định dạng nhớ sửa
  lại 'src' trong src/App.jsx cho khớp phần đuôi file).
- Sau khi copy file vào đây, chạy lại: npm run build

Nhạc chỉ phát khi khách bấm nút "Âm nhạc" (trình duyệt chặn tự động phát).
